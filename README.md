# Anti ISP Complaint Management System

## Overview
A PHP and MySQL based web application for managing ISP customer complaints with separate modules for Admin, Provider, and Customer.

## Features
- User Login & Registration
- Complaint Management
- Admin Dashboard
- Provider Dashboard
- Customer Tracking System
- Service Management

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap

## Installation
1. Install XAMPP
2. Move project folder to htdocs
3. Import database in phpMyAdmin
4. Run localhost project
